User Guide: CNC Probe Reporter

Overview

The CNC Probe Reporter is a tool designed to parse and visualize router probe report files (.txt). It provides a dashboard with job overview, inspection status, Z-height statistics, and detailed data tables.

How to Use

  • Upload Files: Drag and drop your probe .txt files into the designated drop zone or click to browse.
  • Analyze Data: Once uploaded, the app automatically parses the reports and populates the dashboard.
  • Job Overview: View the Job ID and total number of records processed.
  • Inspection Status: A doughnut chart shows the ratio of passed vs. failed parts.
  • Statistics: See average height, range, max, and min values for Z-height (V78).
  • Visualizations: The line chart displays the height distribution across all parts.
  • Data Table: View detailed information for each part, including timestamps and status badges. Outliers are highlighted.

Features

  • Export MD: Download a summary report in Markdown format.
  • Print / PDF: Generate a print-friendly version of the dashboard or save it as a PDF.
  • New Dataset: Clear the current data and start over with new files.
  • Feedback: Use the Feedback button to report bugs or suggest improvements.
